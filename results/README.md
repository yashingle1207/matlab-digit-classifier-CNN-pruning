# Results

This folder stores documented project results and generated visualization outputs from the MATLAB workflow.

## Files

- `metrics_summary.csv` contains the documented before/after metrics from the original coursework notes: validation accuracy, model size, Flash usage, RAM usage, energy per image, and sparsity.
- `ProfilingBeforePruning.png` is the preserved profiling screenshot before pruning-focused optimization.
- `ProfilingAfterPruning.png` is the preserved profiling screenshot after pruning and quantization as documented in the project materials.
- `pruning_accuracy_trend.png` is generated by `src/pruning_quantization_digitsNet.m` when the MATLAB workflow is run.
- `layer_filter_pruning.png` is generated by `src/pruning_quantization_digitsNet.m` when the MATLAB workflow is run.

## Notes

The profiling metrics are documented project results. They may vary with MATLAB version, toolchain version, dataset split, calibration data, compiler/build settings, and the exact EFM32GG11/Simplicity Studio setup used for profiling.
